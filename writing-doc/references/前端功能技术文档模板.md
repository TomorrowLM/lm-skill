## 性能

| 指标             | 目标值   | 说明                 |
| ---------------- | -------- | -------------------- |
| 首屏加载         | < 2s     | 弱网环境首屏可交互时间 |
| 接口响应         | < 500ms  | P95 接口耗时          |
| 大列表渲染       | < 100ms  | 虚拟滚动或分页加载    |

## 兼容性

| 维度           | 要求               | 说明                     |
| -------------- | ------------------ | ------------------------ |
| 浏览器         | Chrome 90+         | 企业内网主流版本         |
| 移动端         | 微信小程序基础库 2.25+ | 最低兼容版本           |
| 分辨率         | 1280×720 ~ 1920×1080 | 主流桌面分辨率        |
