# Git Skills

本目录包含 Git 相关的 AI 技能，所有技能均以 `git-` 前缀命名。

## 结构

```
git/
├── git-commit-conventions/   (Git 提交规范)
├── git-workflow/             (Git 工作流规范)
├── git-finishing-development-branch/ (分支收尾流程)
└── git-worktrees/            (Git Worktree 管理)
```

## 技能列表

| 技能 | 描述 |
|------|------|
| `git-commit-conventions` | Git 提交规范 — 适配国内团队的 commit message 规范和 changelog 自动化 |
| `git-workflow` | Git 工作流规范 — 适配国内平台和团队习惯（Gitee、Coding、极狐 GitLab、CNB） |
| `git-finishing-development-branch` | 完成开发分支 — 引导开发工作的收尾（合并、PR、清理） |
| `git-worktrees` | Git Worktree 管理 — 创建隔离的工作区 |

## 跨平台兼容

所有技能为物理目录，无需符号链接，天然支持 Windows/macOS/Linux 跨平台使用。
